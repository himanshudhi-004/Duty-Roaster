# VIP Visit Form (Frontend Only)

This is a React frontend project that collects VIP visit details through a professional form.

## Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the dev server:
   ```bash
   npm start
   ```

## Notes
- This is frontend-only. There's a placeholder `requirements.txt` file for future Python backend dependencies.
- To connect to a backend, update `src/config.js` with your API URL and implement API calls.
